# OpenTweet

AI-powered tweet generator.  
100% front-end, BYOK.  
Part of the *localhosted.io* tool suite.

## Features
- Generate 1 / 3 / 5 tweets
- EN or FR output
- Markdown-only result
- 100% client-side
- No backend, no tracking

## Requirements
- OpenRouter API key: https://openrouter.ai/

## Self-hosting
Place this folder anywhere:

/tools/opentweet/
index.html
style.css
app.js
README.md
